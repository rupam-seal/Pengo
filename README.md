# Pengo
Pengo is a customer management app

## Tech Stack

- Language: Python, HTML, CSS
- Framework: Django, Bootstrap
- Database: SQL

## Features

- Login, Logout, Registration, Forgot Password
- Admin site, Client site
- Admin can Create, Read, Update and Delete item from database
- User can set profile picture

## Installation

Clone the project

```bash
  git clone https://github.com/rupam-seal/Pengo.git
```

Go to the project directory

```bash
  cd pengo
```

Install Dependencies

```bash
  pip install -r requirements.txt
```

Run django server

```bash
  python manage.py runserver
```
